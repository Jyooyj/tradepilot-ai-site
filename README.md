# TradePilot AI｜拿货搭子

会展选品与爆款测款智能体网站原型。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## Vercel 部署参数

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## 项目亮点

- 产品信息采集
- AI 爆款潜力评分
- 利润测算
- 供应商风险诊断
- 小红书/抖音测款内容生成
- 选品决策报告生成
